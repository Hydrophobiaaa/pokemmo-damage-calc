# PokeMMO Damage Calculator (Raid Focus)

This project is the [Smogon damage calculator](https://github.com/smogon/damage-calc), adapted for PokeMMO with raid focused workflows and pages.

## What This Repository Contains

- A static web calculator app for standard battles and raid scenarios.
- A shared damage engine in `src/calc/`.
- Browser UI controllers and data in `src/js/`.
- Deep dive documentation in `docs/`.

## Documentation

Start with `docs/README.md` for architecture notes and implementation deep dives.
